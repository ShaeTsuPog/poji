Available at [poji.app](https://poji.app)!

## Features
- Cross-platform with no install needed!
- Sync progress across devices using AniList
- High quality image scaling (read why this is important on the [wiki](https://wiki.poji.app/features#image-quality)!)
- Touch support
- Keyboard-friendly UX
- Minimal UI
- Privacy friendly – Poji makes 0 network requests by default, and only to AniList if signed in

Check out the [wiki](https://wiki.poji.app) for more info.
